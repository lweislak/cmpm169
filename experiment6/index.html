Experiment 6 - Grammars & Text Art

Description

This experiment was inspired by ambigrams, words that are still readable from different perspectives. Click anywhere on the canvas in order to generate a random word that will rotate along the x-axis. It can still be read, even when upside down.

Technical

I utilized the Lake Reflection font on each word. I originally wanted to rotate the words along the y-axis instead of the x-axis because that seemed to be what most ambigrams do. However, that would mean that I would have to create original ambigrams for each generated word. It was much more reasonable to use a font that can adapt to any word that I apply it to.

I store about 2,000 words in a .txt file within the experiment folder and then splice a word from that file used the built in random function. The green and blue parts of the RGB color for each word are randomized as well.

Reflection

I was the only contributor to the experiment unless otherwise credited. I am happy with the result of the experiment. The font does a lot of heavy lifting because without it I would simply be displaying and rotating words.

The most difficult part of the experiment was trying to find inspiration. I kept starting and restarting the experiment using different sources of inspiration, but I would never make it very far.
